# WebScraper
WebScraper is a tool that will go on a website and collect the data in the DOM by looking for specific class names or id's.

# Setup
1. **Clone the repository**
2. **Install dependencies**
   ```sh
   npm install
   ```
3. **Run the application**
   ```sh
   node server.js
   ```
